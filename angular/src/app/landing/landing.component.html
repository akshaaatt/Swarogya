Design System

Swarogya is an Automated System which uses automatic report generation, drastically reducing the manual man power needed, resulting in big saving of running cost on every month basis. The system ensures timely attention of the needy patients.

Broad Overview What do we aim to solve?

With the Pandemic of health problems, more and more people are getting affected. This is causing a lot of stress to the hospitals and actuate shortage of Health care professionals. To address this issue, an Automated System is proposed which uses patient skills and technology to tackle this situation and revolutionize the medical industry.
